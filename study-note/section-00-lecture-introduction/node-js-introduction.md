## nodeJS란?
- 오픈소스
    - 누구나 볼 수 있도록 소스코드가 공개 되어 있다.
    - [node GitHub](https://github.com/nodejs/node)
- 크로스 플랫폼
    - 여러 OS에서 사용가능.
        - MacOS, Windows, Linux 등에서 실행 가능
- 자바스크립트 런타임
    - 자바스크립트를 실행할 수 있다.
        - TS 포함
